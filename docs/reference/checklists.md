# Checklists

ARIA includes 7 validation checklists in `_aria/shared/checklists/`. Agents use these during quality gates to ensure completeness.

| Checklist | Used By | Purpose |
|---|---|---|
| `prd-checklist.md` | Maestro | Validate PRD completeness and quality |
| `create-story-checklist.md` | Tempo | Validate story has all required fields |
| `dev-story-checklist.md` | Riff | Validate implementation completeness |
| `code-review-checklist.md` | Pitch | Structured code review criteria |
| `sprint-planning-checklist.md` | Tempo | Sprint planning completeness |
| `correct-course-checklist.md` | Tempo | Course correction assessment |
| `document-project-checklist.md` | Verse | Documentation quality validation |
