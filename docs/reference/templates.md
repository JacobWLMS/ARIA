# Templates

ARIA includes 16 document templates in `_aria/shared/templates/`. Templates provide consistent structure for documents and work item descriptions.

## Project Templates

| Template | Used By | Purpose |
|---|---|---|
| `product-brief-template.md` | Cadence | Executive product brief |
| `prd-template.md` | Maestro | Product Requirements Document |
| `epics-template.md` | Maestro | Epic/story breakdown structure |
| `ux-design-template.md` | Lyric | UX design specification |
| `architecture-decision-template.md` | Opus | Architecture decision document |
| `readiness-report-template.md` | Opus | Implementation readiness assessment |
