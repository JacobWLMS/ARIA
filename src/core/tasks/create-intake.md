# Create Intake

**Purpose:** Add an idea or suggestion to the platform's triage queue. On platforms with native intake support, creates an intake item. On platforms without it, creates a backlog work item instead.

**Parameters:** title (required), description (required), source (optional, e.g., "brainstorming", "user_feedback")

## Execution
1. READ the platform-specific task at {project-root}/_aria/platform/tasks/create-intake.md
2. FOLLOW its instructions with the parameters above
